== Description ==

Create beautiful guides that help your visitors finding the right content.

**Use your very own data** You can use your WordPress data to create all types of filters for your guide.

**Performance** You can create guides out of thousands of posts with no performance issue.

**Show your guides everywhere** You can show your guides with a simple shortcode or the Gutenberg GuidePlugin-Module.

== Installation ==

From your WordPress dashboard

1. **Download** From the link we send you on purchase
2. **Visit** Plugins > Add New > Upload Plugin
3. **Upload** the ZIP file
4. **Activate** GuidePlugin from your Plugins page
5. **Read** the documentation to [get started](https://www.guideplugin.com/documentation/getting-started/getting-started)
